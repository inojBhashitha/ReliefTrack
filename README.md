# ReliefTrack

Smart Disaster Relief Coordination System

## Overview

ReliefTrack is a Java-based disaster relief coordination system developed for the Programming Data Structures and Algorithms (PDSA) coursework.

The system helps relief organizations manage emergency resources efficiently using advanced data structures.

## Primary Data Structure

- AVL Tree

## Supporting Data Structures

- Graph
- Priority Queue
